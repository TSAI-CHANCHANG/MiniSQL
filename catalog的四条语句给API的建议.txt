原则：indexmanager给我，工作我来做


这四个返回值都是int，值为不是SUCCESS都是报错！


Create table 
调用catalog的CreateTable(string Tablename, string Attributes,BT)
得到返回值之后有primary的再调用Index的CreateIndex

目前可以处理的错误：
表重名
表格式错误（包括内容格式错误、类型错误、约束错误）（多个空格OK，换行OK）
两个primary key
primary key单独列在不存在的attribute上

Drop table
调用catalog的DropTable(string Tablename,BT)

目前可以处理的错误：
表不存在

Create Index
调用index的Createindex
调用catalog的CreateIndex(string Indexname, string Tablename, string Attrname,BT)

目前可以处理的错误：
表不存在
Attribute不存在

Drop Index
调用catalog的DropIndex(string Indexname, string Tablename,BT)

目前可以处理的错误：
表不存在
索引不存在
